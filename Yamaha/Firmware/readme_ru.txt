Версия прошивки контроллера ATTiny2313/4313 для работы
одиночной платы kb4y805 в версии c PS/2 клавиатурой в
режиме эмуляции внешней клавиатуры компьютера YIS805.
PS/2 клавиатура работает в режиме Scancode 2.
; Служебные клавиши:
; SELECT -  F11
; STOP   -  F12
; CLS/HOME - Home
; INS - Ins
; DEL - Del
; GRAPH  -  Alt левый и правый
; РУС    -  Scroll Lock
Две версии прошивки для разных версий ПЗУ в YIS805
kbd2yis805_SU.hex - для ПЗУ с набором цифр при нажатом SHIFT
kbd2yis805_RU.hex - для ПЗУ с набором цифр без нажатия SHIFT
